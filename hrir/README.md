# HRIR Files

EAC_Default.wav HRIR preset in two sample rates for HeSuVi.

| File | Sample Rate |
|------|-------------|
| `EAC_Default.wav` | 48 kHz |
| `44/EAC_Default.wav` | 44.1 kHz |

These files are automatically downloaded by the install script during setup. They are included in this repo for reference.

If adding manually, copy the contents of this folder into `C:\Program Files\EqualizerAPO\config\HeSuVi\hrir\`.
