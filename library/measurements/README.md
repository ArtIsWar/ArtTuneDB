# Measurements

Frequency response measurements for headphones and IEMs not widely available on [squig.link](https://gadgetrytech.squig.link).

## Usage

1. Open the `GadgetryTech SquigLink.url` shortcut in the Game/Season folder
2. Upload the target file (e.g., `BF6_Target_S0.txt`) in the **EQ** tab
3. Instead of searching for your headphone, upload the matching measurement file from this folder
4. Hit **Auto EQ** and save the result to the `eq/` subfolder as usual

Still can't find them? Try [AutoEQ Measurements](https://github.com/jaakkopasanen/AutoEq/tree/master/measurements).

## File Format

Each file contains tab-separated frequency (Hz) and amplitude (dB) pairs in squig.link format.
